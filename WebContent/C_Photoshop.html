포토샵 사진 저장




가로, 세로, 길이 등을 조정하고 create 버튼을 누릅니다.




본인이 원하는 이미지 작업 후에 File - Save As 를 선택합니다.




저장하고 싶은 곳과 이름을 설정하고 저장합니다.
파일 형식은 무조건 JPG 혹은 PNG로 저장합니다. (이미지를 저장해야하기 때문)




저장한 위치에 이미지가 있는 것을 확인합니다.
